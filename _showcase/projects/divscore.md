    <h2 data-en="DivScore: Zero-Shot LLM Detection in Specialized Domains" data-zh="DivScore: 专业领域中的零样本大语言模型检测">DivScore: Zero-Shot LLM Detection in Specialized Domains</h2>

            <p data-en="A zero-shot detection framework for identifying LLM-generated text in specialized domains like medicine and law, using normalized entropy-based scoring and domain knowledge distillation." data-zh="一个零样本检测框架，用于识别医学和法律等专业领域中的大语言模型生成文本，使用归一化熵评分和领域知识蒸馏。">

                </li>
                <li data-en="<strong>Normalized entropy scoring</strong>: Robust metric for specialized text" data-zh="<strong>归一化熵评分</strong>：专业文本的稳健指标">
                    <strong>Normalized entropy scoring</strong>: Robust metric for specialized text
                </li>
                <li data-en="<strong>Domain knowledge distillation</strong>: Leverages domain-specific patterns" data-zh="<strong>领域知识蒸馏</strong>：利用领域特定模式">

                </li>
                <li data-en="<strong>Cross-domain robustness</strong>: Tested on medical, legal, and financial texts" data-zh="<strong>跨领域鲁棒性</strong>：在医学、法律和金融文本上测试">
