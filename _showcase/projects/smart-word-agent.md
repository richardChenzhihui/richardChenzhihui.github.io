                </li>
                <li data-en="<strong>Multi-document context</strong> with Kimi-K2 long-context handling for complex multi-section documents" data-zh="<strong>多文档上下文</strong>，利用 Kimi-K2 长上下文能力处理复杂多章节文档">

                </li>
                <li data-en="<strong>Stateless agent loop</strong> with streaming responses for low-latency interaction" data-zh="<strong>无状态 Agent 循环</strong>，流式输出实现低延迟交互">
